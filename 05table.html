테이블 관련 태그

개요

여러종류 테이블을 일목요연하게 2차원형태로 표로 나타내기 위해 사용하는 요소

table태그로 테이블시작을 tr태그로 테이블 행을 th태그로 테이블의 제목을 td태그는 테이블의본문을 나타

사원 employees 테이블

테이블의 디자인 관련 속성은 css를 이용(width, height, padding, border-collapse)

복잡한표 만들기

복잡한 테이블 작성하려면 colspan/rowspan 속성을 사용

colspan 여러열 을 합쳐 하나의 열로 rowspan은 여러 행을 합쳐 하나의 행으로

테이블 너비 지정

테이블 요소의 크기를 지정하는 단위는 px,%임

크기를 픽셀로 지정하는 경우 고정크기로 설정되어 브라우저의 크기에 상관업시이 테이블의 크기는 변화 없음

% 로 지정하는 경우가변크기로 설정되어 브라우저의 크기에 따라 테이블의 크기도 변함
