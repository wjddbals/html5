시맨틱 웹관련 요소

개요

시맨틱 태그란 브라우저, 검색엔진, 개발자 모두에게 콘텐츠의 의미를 명확히 설명하는 역할 담당 시맨틱 태그에 의해 컴퓨터가 HTML 요소의 의미를 보다 명확히 해석하고 데이터로 활용할 수 있는 시맨틱 웹을 실현할 수 있음

시맨틱 웹이란 웹에 존재하는 수많은 웹페이지들에 메타데이터(Metadata)를 부여하여, 기존의 잡다한 데이터 집합이었던 웹페이지'의미'와 '관련성'을 가지는 새로운 정보를 만들어 낼 수 있도록만든 차세대 지능형 웹

시매틱 태그 구성

header :웹사이트로고 검색폼 을 나타내믄 영역

nav :웹 사이트를 구성하는 메튜gnb를 나타내즌 영역

main :웹사이트의 중요 컨텐츠를 표시하는 영역

section : 중요 컨텐츠를 구성하는 하우 컨텐츠를 그룹화할때 사용

article : 중요 컨텐츠들을 구성하는 하위 컨텐츠가 보여지는 영역

aside: 웹사이트 중요 컨텐츠를 관련없는 내용을 표시하는 영역

footer: 웹사이트 저작권이나 관리자 정보등을 나타내는 영역
